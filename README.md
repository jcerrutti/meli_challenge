# Meli Challenge

### Author

Juan Cerrutti - jicerrutti@gmail.com

## Instructions

```
$ git clone https://github.com/jcerrutti/meli_challenge.git
$ cd meli_challenge
```

### Middleware

```
$ cd meli-middleware
$ npm install
$ npm start
```

### Web App

```
$ cd meli-webclient
$ yarn install
$ yarn start
```

- You can use npm also
